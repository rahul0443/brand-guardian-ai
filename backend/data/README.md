# Data Sources

Place the following files in this directory before running `index_documents.py`:

1. `1001a-influencer-guide-508_1.pdf` — FTC Endorsement Guidelines
   → Download: https://www.ftc.gov/system/files/documents/plain-language/1001a-influencer-guide-508_1.pdf

2. `youtube-ad-specs.pdf` — YouTube Advertising Specifications
   → Download from YouTube's advertising policies page
